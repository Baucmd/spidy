### info araignées sauteuses

Régime alimentaire : Les araignées sauteuses sont des prédateurs généralistes au régime alimentaire composé de proies variées : petits insectes comme les moustiques ou encore les mouches.

Prédateurs : Cependant, les guêpes, les lézards et les oiseaux sont les principaux prédateurs des araignées.

Lieu de vie : La plupart des salticides vivent dans les forêts tropicales. On les trouve aussi dans les forêts tempérées, les déserts, les montagnes et les zones situées entre les limites de marées hautes et de marées basses. Ces araignées habitent tous les continents sauf les pôles.
